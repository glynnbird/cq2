Step 2 - Querying

We can query our index by posting some JSON to the the /movies/_find endpoint. In curl this would look like this: 

      curl -X POST https://myaccount.cloudant.com/movies/_find -d '{ "selector": { "year": 2012 }}'

In the above example, we are looking for movies released in 2012. Try this query and others by clicking the buttons below. The request and responses will appear below:

  • year = 2012
  • year = 2012 AND rating = 'R'
  • Freetext search for 'Robert de Niro'
  • cast = 'Robert de Niro' ordered by Movie_year

  • year < 2012
  • year < 2012 AND rating != 'R'
  • cast = 'Robert de Niro' or 'Robert Duvall'
  • Freetext match 'Julia Roberts', only two fields
